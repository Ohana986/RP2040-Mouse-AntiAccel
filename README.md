# 利用RP2040中转消除安卓设备上的鼠标加速

**核心简介：**
一种基于 RP2040 的硬件解决方案，通过拦截并补偿 HID 信号，旨在消除安卓设备上的鼠标加速。

**项目说明：**
本项目基于 Arduino IDE 1.8.19 开发，使用了由 sekigon-gonnoc 开发的 Pico-PIO-USB 库。由于目前尚未明确安卓系统具体的鼠标加速算法，现阶段仅实现了信号透传（Passthrough）功能。
想要实现透传，需要给RP2040添加一个usb母口：

* **红色线** -> VBUS 

* **黑色线** -> GND

* **绿色线 (D+)** -> GPIO 0

* **白色线 (D-)** -> GPIO 1

这样鼠标可以通过拓展的usb母口连接到pico，再通过自带的type c口连接到安卓设备

烧录固件，需要按住pico上的BOOTSEL按钮，保持按住状态，把自带type c口连接到电脑，点击arduino的上传即可

## Eliminating Mouse Acceleration on Android via RP2040 Intermediary

**Core Overview:**
A hardware-based solution using the **RP2040** to intercept and compensate for HID signals, specifically designed to eliminate mouse acceleration on Android devices.

**Project Details:**
This project is developed using **Arduino IDE 1.8.19** and utilizes the `Pico-PIO-USB` library developed by **sekigon-gonnoc**. Since the specific mouse acceleration algorithms used by the Android OS have not yet been fully identified, the project currently implements a **signal passthrough** function as a foundation.

To achieve passthrough, you must add a USB-A female port to the RP2040 using the following wiring:

* **Red Wire (VBUS)** -> VBUS
* **Black Wire (GND)** -> GND
* **Green Wire (D+)** -> GPIO 0
* **White Wire (D-)** -> GPIO 1

**Setup:** The mouse connects to the RP2040 via the expanded USB female port, and the RP2040 connects to the Android device via its onboard Type-C port.

**Flashing Firmware:**

1. Press and hold the **BOOTSEL** button on the Pico.
2. While holding the button, connect the onboard Type-C port to your computer.
3. Click **Upload** in the Arduino IDE.
