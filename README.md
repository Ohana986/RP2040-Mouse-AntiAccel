# 利用RP2040中转消除安卓设备上的鼠标加速

核心简介：
一种基于 RP2040 的硬件解决方案，通过拦截并补偿 HID 信号，旨在消除安卓设备上的鼠标加速。

# RP2040-Mouse-AntiAccel
A hardware-based solution using RP2040 to disable mouse acceleration on Android by intercepting and compensating HID signals.

Based on Arduino IDE 1.8.19 and the Pico-PIO-USB library developed by sekigon-gonnoc. Due to the unknown specific mouse acceleration algorithm used by Android, currently only the passthrough functionality is implemented.
